Green municipalities

The Green municipalities project aims to show the importance of performance in websites by showing how a website's size in kilobytes and megabytes affect its carbon footprint.

The numbers on this page are calculated by Website Carbon's API. As can be seen below, each page view does not emit very many grams of CO₂, however these websites only get a tiny portion of the page views of the world, so it definitely adds up.

Norwegian municipalities have been chosen because we think public services should be role models, and because there are more than 300 of them which are highly similar in contents. Do you see your website on the list and you want to improve? For now, please reach out to us by creating an issue on GitHub, either in English or Norwegian.

We can optimize images by scaling them down to just as big as we need them, and there are many techniques to make a website's code base smaller. The Green municipalities project shows how many carbon dioxide equivalents are emitted by opening each of the Norwegian municipalities' websites, and we want to offer tips and tricks for decreasing the emissions each visitor produces.

This project is still in development and can be followed on GitHub. Contributions are welcome.
